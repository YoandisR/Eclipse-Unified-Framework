Características Principales
  • Gestión Avanzada de Aplicaciones del Sistema
  • Análisis de Sistema en Tiempo Real
  • Captura de Pantalla con Opciones Avanzadas
  • Control Completo del Dispositivo
  • Gestión Inteligente de Caché del Sistema
  • Historial de Comandos con Búsqueda Avanzada
  • Interfaz Estilizada Tipo "Hacker"
  • Soporte Nativo para Root y Shizuku
Módulos Integrados
Gestión de Aplicaciones

Comandos: appmanager, pm
Instalación, desinstalación y gestión completa de aplicaciones del sistema

Sistema y Dispositivo

Comandos: sysinfo, device, process
Información detallada del sistema y control avanzado del dispositivo

Utilidades Avanzadas

Comandos: screenshot, cache, history
Herramientas esenciales para administración profesional

Lanzamiento de Apps

Comando: open
Lanza aplicaciones predefinidas o personalizadas con parámetros

Comandos de Ejemplo

python3 eclipse.py run sysinfo # Información completa del sistema

python3 eclipse.py run screenshot --delay 3 # Captura con retardo

python3 eclipse.py run process --sort cpu --limit 10 # Top 10 procesos por CPU

python3 eclipse.py run cache clear-all # Limpieza total de caché

python3 eclipse.py interactive # Modo interactivo avanzado

python3 eclipse.py run open --app whatsapp # Abrir WhatsApp

python3 eclipse.py ui # Abrir interfaz gráfica (NUEVO)

Requisitos del Sistema
  • Android 7.0 o superior (API 24+)
  • Termux instalado y configurado
  • Acceso Root o Shizuku (para funciones avanzadas)
  • Python 3.7+ (incluido automáticamente en Termux)
  • Dependencias: click, psutil, requests, colorama (auto-instalación)
  • Espacio libre: mínimo 50MB para instalación completa
